Dispositivos Móviles

Repositorio de prácticas del curso de Dispositivos Móviles.
Los ejercicios estan en el lenguaje Kotlin utilizando scripts independientes por cada problema.

## Autor: Cinthia Blanco

## Organización del repositorio

El repositorio está organizado por prácticas:

```
dispositivos-moviles/
 └── practica1/
      ├── evaluacion_empleados.kt
      ├── piedra_papel_tijera.kt
      ├── calculadora.kt
      └── adivina_numero.kt
```

## Descripción de la Práctica 1

* Evaluación de empleados: Calcula el nivel de rendimiento y el dinero recibido según la puntuación.
* Piedra, Papel o Tijera: Juego contra la computadora con elección aleatoria.
* Calculadora elemental: Permite realizar operaciones básicas mediante un menú.
* Adivina el número: Juego donde el usuario intenta adivinar un número aleatorio con intentos limitados.

## Requisitos

* Kotlin (versión 1.5 o superior)
* JDK instalado

## Cómo ejecutar los programas

1. Descargar o clonar el repositorio:

```
git clone https://github.com/Cinthia0709/dispositivos-moviles.git
```

2. Acceder a la carpeta del proyecto:

```
cd dispositivos-moviles/practica1
```

3. Compilar y ejecutar un archivo Kotlin desde consola:

```
kotlinc nombre_archivo.kt -include-runtime -d programa.jar
java -jar programa.jar
```

(Reemplazar "nombre_archivo.kt" por el archivo que se desea ejecutar)
