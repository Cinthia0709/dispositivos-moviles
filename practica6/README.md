# Práctica 6 - Navegación y Listas en Jetpack Compose

Aplicación Android en Kotlin usando Jetpack Compose.

## Funcionalidades

- Lista de contactos con LazyColumn
- Marcar y desmarcar favoritos 
- Eliminar contactos
- Orden automático: favoritos primero
- Navegación entre pantallas (NavHost)

## Pantallas

- Lista de contactos
- Formulario de registro

## Tecnologías

- Kotlin
- Jetpack Compose
- Navigation Compose
