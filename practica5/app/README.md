# Práctica 5 - Formulario Mejorado con Jetpack Compose

Aplicación Android desarrollada en Kotlin usando Jetpack Compose.

## Funcionalidades

- Registro con campos:
  - Nombre
  - Edad
  - Correo electrónico

- Componentes Compose usados:
  - RadioButton (género)
  - Checkbox (términos)
  - Switch (activo/inactivo)
  - Slider (nivel)

- Validaciones:
  - Nombre obligatorio
  - Edad solo números
  - Correo válido con @
  - Aceptar términos obligatorio

- Botones:
  - Registrar (solo activo si todo es válido)
  - Limpiar formulario

- Resultado final mostrado en pantalla.

## Tecnologías

- Kotlin
- Android Studio
- Jetpack Compose
