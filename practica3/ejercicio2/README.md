# Práctica 3 - Aplicaciones Básicas Android

## Ejercicio 2: Reproductor de Música Básico

Aplicación Android desarrollada en Kotlin que permite reproducir un archivo de audio local mediante el uso de botones interactivos.

### Funcionalidades
- Reproducir audio
- Pausar reproducción
- Detener reproducción

### Tecnologías utilizadas
- Kotlin
- Android Studio
- MediaPlayer

### Estructura
- `MainActivity.kt`: Lógica de reproducción de audio
- `activity_main.xml`: Interfaz de usuario
- `Zoe - Labios Rotos`: Archivo de audio utilizado

### Descripción
Se puede controlar la reproducción de un archivo de audio almacenado en la carpeta `res/raw` mediante botones en pantalla. Se utiliza la clase `MediaPlayer` para gestionar el audio.
